Le HTML (HyperText Markup Language) est le langage standard utilisé pour créer des pages web.

PHP (Hypertext Preprocessor) est un langage de script côté serveur largement utilisé pour le développement web

JavaScript (JS) est un langage de programmation permettant de rendre les pages interactives et dynamiques

Angular est un framework, utilisé pour construire des applications web dynamiques et complexes.

TypeScript (TS) est un surensemble de JavaScript qui ajoute des types statiques à ce dernier

MySQL est un système de gestion de base de données relationnelle (SGBDR) open-source

Blender est un logiciel de 3d qui permet de créer des objet en 3d

Unity est un moteur de jeu puissant et polyvalent, utilisé pour créer des jeux vidéo en 2D et 3D
